# 📝 Imagine Task

Projeto desenvolvido para a disciplina de **Fábrica de Software** do curso de Engenharia de Software da UNIVILLE.

---

## 📌 Sobre o projeto

O **Imagine Task** é um sistema simples de organização de tarefas do dia a dia.

A proposta é ajudar o usuário a manter suas atividades organizadas em um único lugar, evitando esquecimentos e melhorando a produtividade.

---

## 🎯 Objetivo

Permitir que o usuário:

* Crie novas tarefas
* Visualize todas as tarefas em uma lista
* Marque tarefas como concluídas

---

## ⚙️ Funcionalidades

* Cadastro de tarefas
* Listagem de tarefas
* Marcar como concluída (com alteração visual)
